# What
PlayFab SDK for Unity

# How to install

Add the lines below to Packages/manifest.json

for version `2.153.221024`
```csharp
"com.pancake.playfab": "https://github.com/pancake-llc/playfab.git?path=Assets/_Root#2.153.221024",
```
